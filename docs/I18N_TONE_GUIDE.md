# دليل نبرة الترجمة العربية

## الهدف

هذا الدليل يضمن أن النصوص العربية في Deep Solution تبدو طبيعية واحترافية، وليست ترجمة حرفية من الإنجليزية. الهدف هو أن يشعر المستخدم العربي أن المنتج صُمم له من البداية.

---

## القواعد الأساسية

### 1. الإيجاز والوضوح

النص العربي الجيد في الواجهات يكون مختصراً ومباشراً. تجنب الجمل الطويلة والشروحات المفرطة.

| ❌ ترجمة حرفية | ✅ عربية طبيعية |
|---------------|----------------|
| يرجى الانتظار بينما نقوم بتحميل البيانات الخاصة بك | جارٍ التحميل... |
| لقد تم إرسال طلبك بنجاح | تم الإرسال |
| هل أنت متأكد من أنك تريد حذف هذا العنصر؟ | حذف هذا العنصر؟ |
| لا توجد أي نتائج تطابق معايير البحث الخاصة بك | لا توجد نتائج |

### 2. ترتيب الجملة العربية

العربية تفضل ترتيب: فعل + فاعل + مفعول. لا تتبع ترتيب الجملة الإنجليزية.

| ❌ ترتيب إنجليزي | ✅ ترتيب عربي |
|-----------------|--------------|
| المنتج تم إضافته | أُضيف المنتج |
| الطلب تم تأكيده | تم تأكيد الطلب |
| أنت قمت بتسجيل الدخول | سجّلت الدخول |
| هذا الإجراء لا يمكن التراجع عنه | لا يمكن التراجع عن هذا الإجراء |

### 3. تجنب الضمائر الزائدة

العربية لا تحتاج ضمائر كثيرة لأن الفعل يحمل الضمير.

| ❌ ضمائر زائدة | ✅ طبيعي |
|---------------|---------|
| أنت تستطيع أن تقوم بـ | يمكنك |
| هو يقوم بالتحميل | جارٍ التحميل |
| نحن نقوم بمعالجة طلبك | جارٍ معالجة الطلب |
| أنت قمت بإدخال | أدخلت |

### 4. الأفعال بدلاً من الأسماء

استخدم الأفعال المباشرة بدلاً من التراكيب الاسمية الثقيلة.

| ❌ تركيب اسمي | ✅ فعل مباشر |
|--------------|-------------|
| القيام بعملية الحفظ | احفظ |
| إجراء عملية البحث | ابحث |
| القيام بإرسال | أرسل |
| عملية التصدير | صدّر |

## دليل النصوص الصغيرة (Microcopy)

### الأزرار

الأزرار يجب أن تكون أفعال أمر مباشرة ومختصرة.

| النوع | ❌ سيء | ✅ جيد |
|------|-------|-------|
| حفظ | قم بحفظ التغييرات | احفظ |
| إلغاء | إلغاء العملية | إلغاء |
| حذف | قم بحذف هذا العنصر | احذف |
| إرسال | إرسال النموذج | أرسل |
| تأكيد | تأكيد الإجراء | تأكيد |
| تحديث | تحديث البيانات | حدّث |
| إضافة | إضافة عنصر جديد | أضف |
| تصدير | تصدير البيانات | صدّر |
| استيراد | استيراد البيانات | استورد |

### أزرار AI

أزرار الذكاء الاصطناعي تحتاج صياغة خاصة توحي بالمساعدة لا الاستبدال.

| ❌ سيء | ✅ جيد |
|-------|-------|
| توليد تلقائي | اقترح |
| إنشاء بالذكاء الاصطناعي | ساعدني |
| AI Generate | اكتب مسودة |
| تحليل ذكي | حلّل |

### رسائل الخطأ

رسائل الخطأ يجب أن تكون واضحة ومفيدة، لا تقنية أو مخيفة.

| ❌ سيء | ✅ جيد |
|-------|-------|
| حدث خطأ غير متوقع | حدث خطأ، حاول مجدداً |
| فشل في الاتصال بالخادم | تعذر الاتصال، تحقق من الإنترنت |
| خطأ 404: الصفحة غير موجودة | الصفحة غير موجودة |
| حقل مطلوب | هذا الحقل مطلوب |
| تنسيق البريد الإلكتروني غير صالح | البريد الإلكتروني غير صحيح |
| كلمة المرور يجب أن تحتوي على 8 أحرف على الأقل | كلمة المرور قصيرة جداً |

### رسائل النجاح

| ❌ سيء | ✅ جيد |
|-------|-------|
| تمت العملية بنجاح | تم ✓ |
| تم حفظ التغييرات بنجاح | حُفظت التغييرات |
| تم إرسال الرسالة بنجاح | أُرسلت الرسالة |
| تم إنشاء الحساب بنجاح | مرحباً بك! |

### الحالات الفارغة (Empty States)

| السياق | ❌ سيء | ✅ جيد |
|-------|-------|-------|
| لا منتجات | لا توجد أي منتجات في هذه القائمة حالياً | لا توجد منتجات بعد |
| لا طلبات | لم يتم العثور على أي طلبات | لا توجد طلبات |
| لا نتائج بحث | لم يتم العثور على نتائج تطابق بحثك | لا توجد نتائج |
| لا إشعارات | ليس لديك أي إشعارات جديدة | لا توجد إشعارات |

### التأكيدات

| ❌ سيء | ✅ جيد |
|-------|-------|
| هل أنت متأكد من أنك تريد المتابعة؟ | متابعة؟ |
| هل تريد حقاً حذف هذا العنصر؟ | حذف هذا العنصر؟ |
| سيتم حذف جميع البيانات المرتبطة | سيُحذف نهائياً |

## قواعد الأرقام والعملات

### الأرقام

استخدم الأرقام الهندية (٠١٢٣٤٥٦٧٨٩) أو العربية (0123456789) بشكل متسق. نفضل الأرقام العربية للتوافق التقني.

| السياق | المثال |
|-------|--------|
| الكميات | 25 منتج |
| الأسعار | 150.00 ر.س |
| التواريخ | 15 يناير 2024 |
| النسب | 25% |

### العملات

العملة تأتي بعد الرقم في العربية.

| ❌ سيء | ✅ جيد |
|-------|-------|
| SAR 150 | 150 ر.س |
| ر.س 150 | 150 ر.س |
| 150 ريال سعودي | 150 ر.س |

## بنية ملفات الترجمة

### هيكل المجلدات

```
locales/
├── ar/
│   ├── common.json      # نصوص مشتركة
│   ├── dashboard.json   # لوحة التحكم
│   ├── products.json    # المنتجات
│   ├── orders.json      # الطلبات
│   ├── settings.json    # الإعدادات
│   └── errors.json      # رسائل الخطأ
└── en/
    ├── common.json
    ├── dashboard.json
    └── ...
```

### تسمية المفاتيح

استخدم مفاتيح ثابتة ووصفية بالإنجليزية.

```json
{
  "actions": {
    "save": "احفظ",
    "cancel": "إلغاء",
    "delete": "احذف",
    "confirm": "تأكيد"
  },
  "status": {
    "loading": "جارٍ التحميل...",
    "saving": "جارٍ الحفظ...",
    "success": "تم",
    "error": "حدث خطأ"
  },
  "empty": {
    "products": "لا توجد منتجات بعد",
    "orders": "لا توجد طلبات",
    "results": "لا توجد نتائج"
  }
}
```

### قواعد المفاتيح

1. **استخدم snake_case** للمفاتيح: `order_status` وليس `orderStatus`
2. **تجميع منطقي**: اجمع المفاتيح المتعلقة تحت كائن واحد
3. **لا تكرر**: إذا كان النص مستخدماً في أكثر من مكان، ضعه في `common.json`
4. **وصفية**: `btn_save` أفضل من `b1`

## قائمة المراجعة للمترجم

قبل اعتماد أي ترجمة، تأكد من:

- [ ] النص مختصر ومباشر
- [ ] ترتيب الجملة عربي طبيعي
- [ ] لا توجد ضمائر زائدة
- [ ] الأفعال في صيغة الأمر للأزرار
- [ ] الأرقام بتنسيق متسق
- [ ] العملة بعد الرقم
- [ ] لا توجد مصطلحات تقنية غير مترجمة
- [ ] النص يبدو كأنه كُتب بالعربية أصلاً

## ملاحظات للمطورين

1. **لا تستخدم Google Translate** للترجمة الأولية
2. **راجع مع متحدث عربي أصلي** قبل الاعتماد
3. **اختبر في RTL** للتأكد من عدم انكسار التخطيط
4. **استخدم خط IBM Plex Sans Arabic** لضمان وضوح النص
5. **تجنب النصوص الطويلة** في الأزرار والتسميات
